Title: Session anlegen und verwalten
----
URL-Key: session-anlegen-und-verwalten
----
Cat: Edit
----
Steps:

1:
	Subheader: Session anlegen
	Action:  Wählen Sie **Modus &rarr;  Bearbeitung**
	Details: &rarr; Sie gelangen in den Modus **Bearbeitung**.
2:
	Subheader: 
	Action: Klicken Sie in der Kopfzeile auf den **+**-Button.
	Details: &rarr; Ein Dialogfenster wird geöffnet.

3:
	Subheader: 
	Action: Geben Sie einen Namen und ein Kürzel für die Session ein.
	Details: 	

4:
	Subheader: 
	Action: Klicken Sie auf den **Erstellen**-Button.
	Details: &rarr; Die Session wird unter dem angegebenen Namen erstellt und es wird automatisch eine Session-ID generiert.

5:
	Subheader: 
	Action: Wählen Sie **Modus &rarr; Hörsaal (PI)**.
	Details: &rarr; Die Standardansicht (Hörsaal (PI) wird geöffnet.		
		

6:
	Subheader: 
	Action: In der Kopfleiste werden nun Informationen zur Session angezeigt.
	Details: >
			
			- Sessionkürzel im Auswahlfeld
			
			- Session-ID
			
			- QR-Code-Button zur Anzeige des QR-Codes für die Teilnehmer
			
			- Name der Session

			- Anzahl der angemeldeten Session-Teilnehmer

7:
	Subheader: Session auswählen
	Action:  Klicken Sie in das Session-Auswahlfeld in der Kopfleiste und wählen Sie eine Session aus der Liste der verfügbaren Sessions aus.
	Details: >
		 
		 	**Hinweis**: Um die Liste einzugrenzen, geben Sie den/die ersten Buchstaben der Session oder des Sessionkürzels ein. Die Liste wird entsprechend gefiltert.
		
8:
	Subheader: 
	Action: Die Session wird geladen.
	Details:>
		In der Kopfleiste werden Informationen zur Session angezeigt. Falls bereits Feedback zu der Session gegeben wurde, wird das Feedback im Publikumsbereich angezeigt. Im Badge neben dem Namen der Session wird die Anzahl der angemeldeten Teilnehmer angezeigt.
		
9:
	Subheader: Sessions verwalten
	Action: Klicken Sie in das Session-Auswahlfeld in der Kopfleiste und wählen Sie eine Session aus der Liste der verfügbaren Sessions aus.
	Details: &rarr; Die Session wird geladen.

10:
	Subheader: 
	Action: Wählen Sie **Modus &rarr; Bearbeitung**.
	Details: &rarr; Sie gelangen in den Modus Bearbeitung.

11:
	Subheader: 
	Action: Klicken Sie im Dozentenbereich auf den **Session**-Reiter.
	Details: &rarr; Die Session-Informationen werden angezeigt.	

12:
	Subheader: 
	Action: Nehmen Sie die gewünschten Einstellungen vor.
	Details:>
		Sie können
			
						
						- Um den Namen oder das Kürzel der Session zu ändern, geben Sie einen neuen Namen/ein neues Kürzel ein.
			
					 	- Um die Session zu sperren, aktivieren Sie die Checkbox  "Sperren"
			
						- Um die Session zu löschen, klicken Sie auf den **Löschen**-Button und bestätigen Sie anschließend das Löschen im Dialogfeld. **Warnung**: Dieser Schritt kann nicht rückgängig gemacht werden!
			
						- Um die Session in JSON-Format zu exportieren, klicken Sie auf den **Exportieren**-Button.
	
13:
	Subheader: 
	Action: Um die Änderungen zu speichern, klicken Sie auf den **Speichern**-Button.
	Details:  &rarr; Die Einstellungen werden gespeichert.

14:
	Subheader:  QR-Code anzeigen
	Action: Klicken Sie in das Session-Auswahlfeld in der Kopfleiste und wählen Sie eine Session aus der Liste der verfügbaren Sessions aus.
	Details: &rarr; Die Session wird geladen.
		
15:
	Subheader: 
	Action: Klicken Sie auf den **QR-Code**-Button in der Kopfleiste.
	Details: >
		&rarr; Der QR-Code der Session wird in einem fensterfüllenden Overlay angezeigt.
		
			
			**Hinweis**: Um den QR-Code im Vollbild-Modus anzuzeigen, wählen Sie **Ansicht &rarr; Vollbild  &rarr; Gesamte Arbeitsfläche** und klicken dann auf den **QR-Code**-Button.
					
			
		
16:
	Subheader: 
	Action: Um den QR-Code wieder auszublenden, drücken Sie die ESC-Taste oder klicken Sie einmal mit der Maus auf den QR-Code.
	Details:  &rarr; Der QR-Code wird wieder ausgeblendet.	

17:
	Subheader: Session löschen
	Action: Klicken Sie in das Session-Auswahlfeld in der Kopfleiste und wählen Sie eine Session aus der Liste der verfügbaren Sessions aus.
	Details: &rarr; Die Session wird geladen.

18:
	Subheader: 
	Action: Wählen Sie **Modus &rarr; Bearbeitung**.
	Details: &rarr; Sie gelangen in den Modus Bearbeitung.

19:
	Subheader: 
	Action: Klicken Sie im Dozentenbereich auf den  **Session**-Reiter.
	Details: &rarr; Die Session-Informationen werden angezeigt.	

20:
	Subheader: 
	Action: Klicken Sie auf den **Löschen**-Button.
	Details: &rarr; Ein Dialogfenster wird geöffnet.

21:
	Subheader: 
	Action: Um das Löschen der Session zu bestätigen, klicken Sie auf den **Löschen**-Button.
	Details: &rarr; Die Session wird gelöscht.	

22:
	Subheader: 
	Action: Das Löschen der Session wird durch eine Meldung bestätigt.
	Details:
