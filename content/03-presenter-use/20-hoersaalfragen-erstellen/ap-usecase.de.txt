Title: Hörsaalfragen erstellen
----
URL-Key: hoersaalfragen-erstellen
----
Cat: Edit
----
Steps:

1:
	Subheader: 
	Action: Klicken Sie in das Session-Auswahlfeld in der Kopfleiste und wählen Sie eine Session aus der Liste der verfügbaren Sessions aus.
	Details: &rarr; Die Session wird geladen.

		
2:
	Subheader: 
	Action: Wählen Sie **Modus &rarr; Bearbeitung**.
	Details: >
		&rarr; Sie gelangen in den Modus *Bearbeitung*. Im Hauptbereich rechts können Sie nun Fragen editieren.


3:
	Subheader: 
	Action: Wählen Sie im Reiter *Neue Frage* im Auswahlfeld Fragetyp die Option "Hörsaal".
	Details: >

4:
	Subheader:
	Action: >
		Geben Sie ein Thema und die Frage ein.		
			
			
			- **Thema**: Den Themenbereich Ihrer Frage. 
				
				**Hinweis**: Themen der aktuellen Session werden in der Datenbank gespeichert. Wenn Sie in der aktuellen Session bereits Themen erstellt haben, können Sie ein Thema aus dem Auswahlfeld auswählen.
					
		
			- **Frage**: Die Frage selbst.
		
		
	Details:
		
5:
	Subheader: 
	Action: Wählen Sie ein Frageformat aus.		
	Details:>
		In diesem Beispiel wählen Sie Mehrfachauswahl. Die folgenden Fragetypen stehen zur Auswahl&#58;
		
		
				- Mehrfachauswahl
				
				- Einfachauswahl
				
				-  Ja/Nein
				
				- Freitext
				
				- Likert Skala				
				
		
			
			Die folgenden Fragetypen können derzeit nur über ARSnova Mobile erstellt werden:		
				
				
				- Note (Schulnotensystem)
				
				- Lernkarte (zum Selbststudium)
				
				- Planquadrat

6:
	Subheader: 
	Action:>
		Geben Sie eine Antwortoption ein und klicken Sie auf den **Hinzufügen**-Button und drücken Sie die Eingabe-Taste. 
		
		**Hinweis**: Die folgenden Antwortmöglichkeiten variieren je nach gewähltem Fragetyp.		
	Details:> 
		Wiederholen Sie diesen Schritt für jede Antwortoption.
		
		&rarr; Die Antwortoptionen werden unterhalb des Eingabefeldes angezeigt.

7:
	Subheader: 
	Action: Markieren Sie die richtige(n) Antwort(en).
	Details: 
				
8:	
	Subheader: 
	Action: Aktivieren bzw. deaktivieren Sie die folgenden Frageoptionen.
	Details: >
		
			- **Enthaltungen erlauben**: Standardeinstellung ist "deaktiviert"
			
			- **Frage freischalten**: Standardeinstellung ist "aktiviert"
			
			- **Statistik einsehbar**: Standardeinstellung ist "aktiviert"
			
			- **Korrekte Anwort einsehbar**: Standardeinstellung ist "deaktiviert"

9:	
	Subheader: 
	Action: Um die Frage zu erstellen, klicken Sie auf den **Erstellen**-Button.
	Details:	&rarr; Die Frage wird gespeichert.

10:	
	Subheader: 
	Action: Eine Erfolgreichmeldung zeigt, dass die Frage gespeichert wurde. 
	Details:	&rarr; Die Frage wird im Dozentenbereich im Reiter **Hörsaalfragen** gelistet.

11:
	Subheader:
	Action: Um weitere Fragen zu erstellen, klicken Sie auf den **+**-Reiter.
	Details: &rarr; Ein neuer Frage-Reiter wird geöffnet.
	
12:
	Subheader:
	Action: Wenn Sie mehr Frage-Reiter geöffnet haben als im Publikumsbereich Platz finden, können Sie mit den Pfeilen nach rechts und links navigieren. Mit dem Pfeil nach unten öffnen Sie ein Menü, mit dem Sie ebenfalls gezielt auf einzelne Fragen zugreifen bzw. neue Fragen hinzufügen können.
	Details:
