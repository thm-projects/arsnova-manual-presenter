Title: Create lecture questions
----
URL-Key: create-lecture-questions
----
Cat: Edit
----
Steps:

1:
	Subheader: 
	Action: Click into the Session select field in the header bar and select a session from the list of available sessions. 
	Details: &rarr; The session is loaded.

		
2:
	Subheader: 
	Action: Go to  **Mode &rarr; Editing**.
	Details: >
		&rarr; You get into *Editing* mode. In the students' pane on the left of the main window you can now create questions. 


3:
	Subheader: 
	Action: In a *New Question* tab, select  *Lecture* from the question type options in the select field.
	Details: 

4:
	Subheader:
	Action: >
		Enter a topic and a question.		
			
			
			- **Topic**: The topic of your question. 
				
				**Note**: Topics of the current session are stored in the database. If you have already created one or more topics, you can select a topic from the select field.
					
		
			- **Question**: The text of your question.
		
		
	Details:
		
5:
	Subheader: 
	Action: Select a question format from the select field.		
	Details:>
		In this example, let's create a multiple choice question. The following question types are available&#58;
		
		
				- Single Choice
				
				- Multiple Choice
				
				- Yes/No
				
				- (Likert) scale

				- Free text		

				
		
			
			Currently, the following question types can only be created using ARSnova Mobile:
				
				
				- **A-F**: Grade
				
				- **FC**: Flashcard
				
				- **P**: Picture Question
		
6:
	Subheader: 
	Action:>
		Enter an answer option and click on the **Add** button or hit the Enter key.
		
		**Note**: The possible answers  depend on the question format you selected.		
	Details:> 
		Repeat this step for each answer option that you want to add. 
		
		&rarr; The answers are listed below the input field.
		

7:
	Subheader: 
	Action: Check the correct answer(s).
	Details: 
				
8:	
	Subheader: 
	Action:>
		Activate/deactivate the following options:
	Details: >
		
			- **Allow abstensions**: default is  "not active"
			
			- **Release question**: default is "active"
			
			- **Release statistics**: default is "active"
			
			- **Release correct answer**: default is "not active"

9:	
	Subheader: 
	Action: To save the question, click on the **Create** button.
	Details:	&rarr; The question is saved.

10:	
	Subheader: 
	Action: A success message confirms that the question was saved. 
	Details:	&rarr; The question is listed in the  **Questions for lecture** tab.

11:
	Subheader:
	Action: To add another question, click on the **+** tab.
	Details: &rarr; A new question tab is opened.
	
12:
	Subheader:
	Action: If you have opened more *New Question* tabs than can be accomodated in the students' pane, you can navigate between them using the arrows to the right and left.  When you click on the down arrow, a menu is opened from which you can select a single question or add new questions.
	Details:
